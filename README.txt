# display-csv-file Git repository

Display a csv file in a web browser.
Useful when there is no spreadsheet tool installed on your system.

A very simple project to check my knowledge from OpenClassrooms "GÃ©rer votre code avec Git et GitHub" part 2 course.

------------
Ressources :
------------

- htmltag.py          : html framework.
- display-csv-file.py : code.
- example.sh          : run this script to see an example of use.
- csv-test-file.csv   : data file used in the example.
- csv-test-file.css   : style sheet.
- README.txt          : project overview.

----------------
Project stages :
----------------

1 - Prepare an html table from the csv file to test the html framework :
    - import framework, read csv file, return an html table.
    - run example.sh to check the results.

2 - Prepare an html page :
    - create head and body, add html table above, write it to a file.
    - run example.sh and open "display-csv-file.html" in a web browser.

3 - Add css :
    - add class properties to each table cell value, add css ref to html page.
    - run example.sh and open "display-csv-file.html" in a web browser.

4 - Add title, start the web browser :
    - run example.sh that opens "display-csv-file.html" in a web browser.
